Bower-ui

A simple UI for the bower package manager

View on GitHub Download .zip Download .tar.gz

This is a humble attempt to build a GUI around bower. At it's core it lives from the bower terminal command line tool. Reading the output that you'd see in the terminal and converting it to a graphical interface.
